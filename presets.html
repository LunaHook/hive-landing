Presets & Configs

Themes that feel like yours.

Import any of these starter presets into HIVE in seconds, or use them as a jumping-off point for your own custom themes.

How to import a preset into HIVE Copy the config ID below, then in HIVE open
Settings → Themes → Import Config
and paste the ID. The theme will load instantly.
