What's inside

Built to keep many sessions calm

Everything you need to run a swarm of OpenCode sessions without losing the thread.

Session management

Create, open, archive, and restore OpenCode sessions in a click. Pin the important ones, tag and group the rest, and jump anywhere with the quick switcher or broadcast a prompt to them all at once.

Live CPU & RAM insights

Watch every session breathe. Real-time CPU and memory graphs, per-session process trees, plus cost, token, and activity trends — so you always know what's running hot.

Theme customization

Make it yours with rich, editable themes and a deep-space palette out of the box. An optional dynamic dock icon recolors itself to match your active theme — down to the accent glow.

⬢

Built for reliability

A hardened session and terminal lifecycle, guarded polling that survives sleep/wake cycles, and no runaway processes. HIVE stays fast and steady on long, marathon days.

Insights

A live pulse on everything

HIVE reads straight from your OpenCode data and each session's process tree to chart CPU and memory in real time. Spot the session that's pinning a core, track spend and token burn over the day, and see your activity heatmap at a glance.

  • Per-session CPU & memory, updated live
  • Cost & token trends by day, week, and month
  • Activity heatmap across every session
